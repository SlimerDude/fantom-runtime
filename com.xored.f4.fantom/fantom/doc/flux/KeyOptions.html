const class

flux::KeyOptions

sys::Obj
  flux::KeyOptions

@Serializable { simple=false collection=false }

KeyOptions allow the user to redefine accelerators for the flux command set.

Slots

bindingsSource

const Str:Str bindings := [Str:Str][:]

Binding of command ids to key accelerators. The keys of this map are FluxCommand.ids. See CommandId for the commonly used predefined commmands. The values of the map are string representations of fwt::Key. If a command is not mapped in this table, then it defaults to the accelerator defined by the command's localized props.
