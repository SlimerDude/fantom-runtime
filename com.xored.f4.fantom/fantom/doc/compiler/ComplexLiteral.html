class

compiler::ComplexLiteral

sys::Obj
  compiler::Node
    compiler::Expr
      compiler::ComplexLiteral

ComplexLiteral is used to model a serialized complex object declared in facets. It is only used in facets, in all other code complex literals are parsed as it-block ClosureExprs.
